SeeQC #QC1000A PDK

This package constitutes the official design kit for the SeeQC QC1000A Process.

Here is what you should do:

Contents

Technology File

A technology file is included which contains all of the necessary physical layers included in the design rules. Please consult the official design rules for a description of the layer stack.

DRC

DRC script is inclduded in order to verify your design before submission.

Libraries

A standard cell library in included. It contains the following cells:

  • DC-SFQ Converter
  • SFQ-DC Converter
  • JTL

More cells may be added in the future.
